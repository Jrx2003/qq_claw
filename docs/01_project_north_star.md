# 01 项目北极星

## 项目名称
虾局长

## 赛道
命题赛道题目 3：QQ 养虾官方 Agent

## 我们对题目的解释
不是“做一个会聊天的虾”，而是做一个 **会推进群内社交行动的官方 Agent**。

换句话说：
- 它不是陪聊 bot
- 它不是万能工具箱
- 它不是普通投票插件
- 它是一个 **把模糊意图收口成真实互动** 的群场景 AI 助手

## 一句话定位
**虾局长是 QQ 群里的社交推进 Agent，负责识别意图、代替人承担组织成本、推动群内从“有人想法”走到“真的成局”。**

## 北极星问题
很多校园群里不是没有想法，而是：
- 有人想约，但不想当组织者
- 讨论很多，但没人收口
- 大家都说“我都可以”，结果什么都没定
- 一次活动结束后，经验和快乐没有沉淀成下次更容易发生的关系资产

## 北极星价值
让虾局长承担“组织者最累但最没人愿意做”的那一段：
1. 识别意图
2. 发起倡议
3. 收口信息
4. 推动投票
5. 确认成局
6. 留下回忆
7. 复用偏好

## 为什么这题适合 AI
如果拿掉 AI，这个产品会明显退化：
- 无法从非结构化群聊里提取真实意图
- 无法把“我都可以”“看情况”“都行”转成可推进决策
- 无法基于上下文自动生成倡议文案、投票、提醒和回忆卡
- 无法在冲突、匿名表达、轻社交等模糊场景中给出“刚刚好”的中介表达

## 当前最强主闭环
宿舍群吃饭局：
群里聊天 → 用户触发虾局长 → 生成成局卡 → 投票 → 成局确认 → 出发提醒 → 回忆卡

## 当前版本的成功标准
不是“接了多少真实接口”，而是：
- 评委一眼能懂产品价值
- 评委 1 分钟内能完成一次交互
- 评委 3 分钟内能看到完整闭环
- 评委觉得“这个东西真的应该出现在 QQ 里”

## 不做成什么
- 不做成泛校园助手
- 不做成自由对话 AI 宠物
- 不做成复杂后台系统
- 不做成“什么都有但什么都不深”的作品
