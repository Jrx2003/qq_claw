# 08 视觉规范（QQ 群聊风格）

## 目标
视觉上让人一眼联想到：
- QQ 群聊
- 校园熟人社交
- 腾讯系产品感
- 移动端真实产品，而不是比赛草图

## 整体风格关键词
- 轻
- 干净
- 克制
- 可信
- 软圆角
- 白底浅灰
- 卡片层次明确
- 信息密度中高，但不拥挤

## 页面结构
### 顶部导航
- 群名居中：`507宿舍干饭群`
- 小号副文案：`8 人在线`
- 左侧返回按钮
- 右侧电话 / 更多图标

### 聊天背景
- 极浅灰
- 消息泡泡白色
- 用户本人气泡可用浅蓝
- Bot 泡泡保持白色，依赖头像和 badge 区分

## 卡片风格
### 通用
- 12~16px 圆角
- 浅描边
- 很轻的阴影
- 头部信息区与内容区分层
- 卡片内边距 16~20px

### 状态色
- 正在收口中：橙色系
- 投票进行中：粉 / 红系
- 已成局：绿色系
- 已完成：浅绿色或米色成功态

### 按钮样式
- 主操作：蓝色线框 / 蓝色填充
- 次操作：灰线框
- 提醒操作：橙色
- 参加 / 成功：绿色

## 字体层级
- 群名：粗体 18~20
- 卡片标题：粗体 20~24
- 状态：12~14
- 摘要：13~14
- 普通消息：16 左右
- 次级说明：12~13

## 头像与标签
### 虾局长
- 固定专属头像
- 旁边有 `群AI助手` 或 `群管` 标签
- 标签用浅黄底/小号字

### NPC
- 头像和昵称要有宿舍群真实感
- 例如：阿凯、小雨、老周、大壮、奶茶不加糖

## 建议的颜色参考
> 仅做方向，不要求逐像素一致

- 页面背景：#F4F5F7 附近
- 普通气泡：#FFFFFF
- 自己消息：浅蓝渐变或实心蓝
- 主蓝：#4A8DFF 附近
- 主绿：#4CAF50 附近
- 警示橙：#FF9F43 附近
- 柔粉：#FEEAEA 附近
- 文字主色：#1F2329
- 次文字：#6B7280

## 间距建议
- 聊天气泡垂直间距：12~16
- 头像到气泡：8~12
- 卡片到说明文案：10~12
- 卡片内 section 间距：14~18
- keyboard 按钮区：12~16

## 微动效建议
- 新消息淡入 + 上滑 8px
- 虾局长 typing 后再出现卡片
- 卡片状态切换可做轻微数值过渡
- 选项点击后出现 pressed / visited 状态

## 演示时最影响高级感的 5 个点
1. 卡片边距是否舒服
2. 状态 badge 是否清晰
3. 头像和昵称是否真实
4. 按钮区是否像原生交互
5. 消息流是否有节奏感
