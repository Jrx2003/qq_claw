# 16 未来扩展说明

## 扩展 A：匿名倡议
### 价值
- 解决“想提议但不想出头”
- 减少发起活动的社交风险
- 让弱意愿更容易被表达

### 产品形态
- 用户点击“匿名发一个想法”
- 虾局长以官方口吻代发倡议
- 群成员看不到真实发起人
- 后续可继续走投票和成局

### 适用场景
- 宿舍约饭
- 社团出游
- 考前一起复习
- 想吐槽但不想点名

## 扩展 B：冲突中继
### 价值
- 不一定要彻底调停冲突
- 先让情绪不过载
- 替双方做诉求转译

### 产品形态
- 虾局长总结分歧点
- 提供温和转述文案
- 提供冷静 / 私聊 / 转述三个操作

### 演示建议
如果时间不够，不做完整流，只做一张“冷静中继卡”。

## 扩展 C：腾讯游戏局
### 价值
- 和 QQ 兴趣群天然契合
- 可以增强题目 3 的生态味
- 让回忆卡更有情绪价值

### 推荐模板
- 王者荣耀开黑局
- 洛克王国组队局

### 不建议
不要把游戏局做成第二主线，只做模板换皮能力即可。
