# 18 待定问题

## 需要你本人最终拍板的 5 件事
1. 虾局长头像风格：更偏官方、还是更偏可爱？
2. Demo 首页要不要做成“直接进入群聊”还是“先有一句话介绍页”？
3. 是否要把匿名倡议做成可点击扩展场景？
4. 是否要把冲突中继做成一张静态卡还是完整小流程？
5. 录屏时是用 Auto Play 还是手动点击更自然？

## 当前默认建议
1. 头像偏官方一点，不要幼龄化
2. 首页极简，最多一句话 + 开始演示
3. 匿名倡议做扩展场景
4. 冲突中继只做一张卡
5. 录屏用 Auto Play，现场答辩再手动点

## 如果时间只够做最少功能
只做：
- 一个群聊页面
- 一条主剧情
- 四张卡
- 自动播放
